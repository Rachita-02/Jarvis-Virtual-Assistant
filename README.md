# Jarvis Virtual Assistant 

A Python-based voice-controlled virtual assistant that can:
- Recognize speech
- Respond using AI (Groq)
- Fetch news using NewsAPI
- Play music and execute commands

## Features
- Speech Recognition
- AI-powered responses
- News headlines
- Music control
- Voice commands
- Web browsing

## Tech Stack
- Python
- SpeechRecognition
- Groq API
- NewsAPI
- pyttsx3 / gTTS


## How to Run
```bash
pip install -r requirements.txt
python Main.py
